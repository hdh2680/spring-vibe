# 목표 (goal)
- 사전 기반 감정분석에서 “중립 과다” 문제를 완화하기 위해
  - `중립`과 `미분류(사전 미매칭)`를 분리해서 결과/차트에 표시한다.
  - 도메인 단어를 계속 보강할 수 있도록 커스텀 사전 파일을 추가한다.

# 핵심 변경사항 (key changes)
- 감정분석 결과 집계 시, 사전 매칭이 0건인 댓글을 `미분류(unknown)`로 분리 집계한다.
  - `overall`/`byHour`에 `unknownCount/unknownPct` 추가
  - `classifiedTotal` 및 `positivePctClassified/neutralPctClassified/negativePctClassified` 추가
- 감정분포 파이/시간별 변화 차트에 `미분류`를 추가 표시했다(시간별은 점선).
- `sentiment_custom.tsv`를 추가하고, 앱 시작 시 기본 사전(`sentiment_lexicon.tsv`) 위에 overlay 로드하도록 `SentimentLexiconLoader`를 개선했다.

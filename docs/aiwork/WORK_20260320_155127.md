# 목표 (goal)
- `youtubeCommentAnalysisView.html`의 "분석수행"을 여러 번 눌러도 매번 재분석이 가능하도록 한다.
- 재분석 시 기존 분석 결과를 삭제하고(동일 `history_id`) 새 결과로 다시 적재한다.

# 핵심 변경사항 (key changes)
- 분석 수행 로직에서 기존 결과가 이미 존재해도 예외로 막지 않고 재수행 가능하도록 변경했다.
- 재수행 시 아래 3개 테이블에서 현재 `history_id`(및 `user_id` 조건) 기준으로 기존 행을 삭제한 뒤 새 결과를 insert 하도록 변경했다.
  - `youtube_comment_analysis_sentiments`
  - `youtube_comment_analysis_top_keywords`
  - `youtube_comment_analysis_topic_groups`
- 화면에서 `analysisRequestedAt` 유무와 무관하게 분석 수행 버튼이 항상 보이도록 변경했다.

# 목표 (goal)
- 감정분석 결과에 감정분포별 예시 댓글(최대 50개)을 함께 적재하고, 결과 화면에서 탭으로 조회할 수 있게 한다.

# 핵심 변경사항 (key changes)
- 감정분석 JSON(`youtube_comment_analysis_sentiments.result_json`)에 `examples`를 추가했다.
  - `examples.positive|neutral|negative|unknown` 각각 최대 50개
  - 각 예시에는 `commentId`, `publishedAt`, `text`, `score`, `matched`, `label` 포함
- 감정분석 탭 하단에 “감정분포별 댓글입니다 최대 50개 제공” 섹션을 추가하고,
  - `부정/미분류/중립/긍정` 탭으로 예시 댓글 테이블을 전환 표시한다.

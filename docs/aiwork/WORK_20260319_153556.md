# 목표 (goal)
`youtube_comment_analysis_histories` 테이블의 legacy 분석 컬럼(`analysis1_*` ~ `analysis5_*`)을 제거하고, 분석 요청 일시 컬럼을 추가한다.  
추가로 댓글 키워드 분석 결과를 DB에 적재할 신규 테이블(Top N 키워드, 주제별 묶음)을 정의하고 `DATABASE_SCHEMA.md`를 최신화한다.

# 핵심 변경사항 (key changes)
- `youtube_comment_analysis_histories`에 `analysis_requested_at` 컬럼을 추가하고, `analysis1_*` ~ `analysis5_*` 컬럼 삭제하는 DDL을 작성했다.
- 키워드 분석 결과 적재용 신규 테이블 2개를 정의했다:
  - `youtube_comment_analysis_top_keywords`
  - `youtube_comment_analysis_topic_groups`
- `YoutubeCommentAnalysisHistory` 도메인/조회 쿼리에 `analysis_requested_at`을 반영했다.
- `docs/DATABASE_SCHEMA.md`에 신규/변경 테이블 스키마를 최신화했다.

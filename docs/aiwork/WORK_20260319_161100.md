# 목표 (goal)
유튜브 댓글 전처리 상세 화면에서 "분석수행" 버튼으로 키워드 분석(Top N, 주제별 묶음)을 실행하고, 결과를 DB 테이블에 적재한다. 분석결과 화면은 만들지 않되, 적재된 결과를 확인할 수 있도록 JSON 조회 엔드포인트를 제공한다.

# 핵심 변경사항 (key changes)
- `POST /users/youtubeComment/analysis/analyze`에서 분석을 수행하고 결과를 아래 테이블에 적재한다.
  - `youtube_comment_analysis_top_keywords`
  - `youtube_comment_analysis_topic_groups`
- 적재 성공 시 `youtube_comment_analysis_histories.analysis_requested_at`을 업데이트해 2회 분석을 방지한다.
- 분석 결과를 확인할 수 있도록 `GET /users/youtubeComment/analysis/result.json?id=...` JSON 응답을 추가했다.

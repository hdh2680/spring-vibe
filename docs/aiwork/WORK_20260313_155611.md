# 작업 로그 (2026-03-13 15:56)

## 작업 배경
- `templates/admin/**` 경로를 사용하던 화면 템플릿을 앞으로는 `templates/html/**` 하위로 통일하기로 함.
- 폴더 이동 후 Controller의 `contentTemplate` 경로 불일치로 인해 뷰를 찾지 못하는 문제가 발생할 수 있어 참조 경로를 함께 정리함.

## 변경 사항
- 템플릿 경로 이동
  - `/src/main/resources/templates/admin/**` → `/src/main/resources/templates/html/admin/**`
- Controller의 `contentTemplate` 경로 수정
  - `"admin/...` → `"html/admin/...`
  - 대상:
    - `/src/main/java/springVibe/dev/admin/users/controller/UsersController.java`
    - `/src/main/java/springVibe/dev/admin/menus/controller/MenusController.java`
    - `/src/main/java/springVibe/dev/admin/roleMenus/controller/RoleMenusController.java`
- 코딩 규칙 문서 보강
  - 화면 템플릿 기본 위치 및 `contentTemplate` prefix 규칙 추가
  - 대상: `/docs/CODING_RULE.md`

## 확인
- `templates/html/admin/**` 하위에 기존 admin 화면 템플릿이 존재하는지 확인
- (권장) `mvn -DskipTests compile`로 컴파일 확인

# 작업 노트 (AI)

## goal
- 유튜브 댓글 조회가 `maxResults=50` 제한으로 한 번에 50개만 가져오는 문제를 해결한다.
- `pageToken` 기반 페이징을 서버에서 자동으로 반복 호출하여, 사용자가 지정한 개수(`limit`)만큼 댓글을 수집할 수 있게 한다.

## key changes
- 댓글 수집 서비스가 `limit`만큼 여러 페이지를 순회하며 댓글을 누적 수집하도록 변경했다.
- UI에서 `가져올 개수(limit)`를 입력할 수 있게 추가했고, 다음 페이지 요청에도 동일한 `limit`를 유지하도록 했다.
- YouTube API 클라이언트에서 `maxResults` 범위(1~100) 검증을 추가했다.
- 단위 테스트로 페이징 반복/nextPageToken 반환을 검증했다.

# 작업 노트 (AI)

## goal
- 유튜브 댓글 “저장” 기능이 브라우저 다운로드를 유발하지 않도록 변경한다.
- 저장이 성공하면 페이지에서 `저장되었습니다.` alert만 표시한다.

## key changes
- `/users/youtubeComment/export`가 파일 다운로드(`ResponseEntity<Resource>`)를 반환하는 대신, 저장 처리 후 동일 화면을 렌더링하도록 변경했다.
- 저장 성공 시 `successMessage` 모델 값을 세팅하고, 화면에서 해당 값이 있으면 `alert()`를 표시하도록 추가했다.

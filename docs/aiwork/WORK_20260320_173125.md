# 목표 (goal)
- 네트워크 분석 그래프가 “이상하게 보이는” 문제를 완화하기 위해,
  - 네트워크 생성 로직의 토큰 처리 편향을 줄이고
  - 그래프에 상위 노드 라벨을 표시해 해석 가능성을 높인다.

# 핵심 변경사항 (key changes)
- 단어 공동출현 네트워크 생성 시, 댓글에서 토큰을 “앞 N개”만 쓰던 방식을 제거하고 **유니크 토큰 집합** 기반으로 집계하도록 개선했다.
  - `maxTokensPerComment` -> `maxUniqueTokensPerComment`로 변경(기본 200)
- 네트워크 SVG 그래프에 docFreq 상위 노드(기본 12개) 라벨을 텍스트로 표시한다.

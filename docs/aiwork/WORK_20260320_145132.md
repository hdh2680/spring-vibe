## 목표 (goal)
- 주제별 묶음(Topic Groups) 군집도가 충분히 오르지 않는 문제를 개선하기 위해, “키워드 묶음 기반” 분류에 2~3회 반복 재할당(간이 EM) 단계를 추가한다.

## 핵심 변경사항 (key changes)
- 주제별 묶음 알고리즘을 `seed-keyword-cooccurrence-v1` → `seed-keyword-cooccurrence-em-v1`로 개선.
- 동작 방식:
  - E-step: 현재 seedKeywords로 댓글을 점수화해 군집 할당
  - M-step: 각 군집에 할당된 댓글 토큰 빈도로 seedKeywords(centroid)를 재계산
  - 위 과정을 최대 3회 반복(이동이 0이면 조기 종료)
- 분석 파라미터 JSON에도 `topicAlgorithm`, `emIterations`를 반영.

# WORK_20260327_175520

## Goal
- `docs/PRD-amazonProduct.md` 기준으로 Amazon Product 목록 페이지를 구현한다.
- 환경변수 플래그로 검색 백엔드를 스위치한다: 활성화 시 ES 검색, 비활성/장애 시 DB LIKE fallback (DevSearch 의도와 동일).

## Key Changes
- `AmazonProduct` JPA 엔티티 및 목록/검색/카테고리 카운트용 Repository 쿼리를 추가했다.
- `AmazonProductService`를 추가했다:
  - 기본은 DB 페이징 목록.
  - 검색어가 있고 `AMAZON_PRODUCT_ES_ENABLED=true`이면 ES에서 title 검색(필요 시 Ollama로 검색어 영문 키워드 변환 시도) 후 asin 순서대로 DB에서 hydrate.
  - ES 비활성/장애 시 DB LIKE로 fallback.
- `/users/amazonProduct/list` Thymeleaf 페이지(카테고리 사이드바 3 : 상품 7 레이아웃, 카드 UI)를 추가했다.
- `application-local.yml`에 `amazon-product.elasticsearch.enabled`(env: `AMAZON_PRODUCT_ES_ENABLED`)를 추가했다.

## Notes / Assumptions
- `amazon_product` 테이블 컬럼은 `amazon_products_ko.csv` 헤더(asin, title, img_url, product_url, stars, reviews, price, list_price, category_id, is_best_seller, bought_in_last_month, product_name_ko)와 일치한다고 가정했다.
- ES 인덱스명은 현재 document annotation에서 `amazon-products`로 고정(dev/local)되어 있다.

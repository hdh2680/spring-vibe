# 목표 (goal)
분석결과 버튼 클릭 시 분석 결과 화면(`youtubeCommentAnalysisView.html`)으로 이동하고, 화면에서 키워드 분석(Top N / 주제별 묶음) 결과를 탭 형태로 시각화해서 보여준다. 또한 목록 화면(`youtubeCommentAnalysis.html`)에서도 분석일시(`analysis_requested_at`)가 존재하면 분석결과로 이동할 수 있게 한다.

# 핵심 변경사항 (key changes)
- `youtubeCommentHistoryView.html`의 "분석결과" 버튼을 JSON 링크가 아닌 분석결과 화면으로 이동하도록 변경했다.
- 분석결과 화면 `youtubeCommentAnalysisView.html`을 추가했다.
  - 탭: "키워드 분석"(Top N bar + 주제 분포 스택 + 주제 카드), "원본 JSON"(debug)
  - 결과 데이터는 `GET /users/youtubeComment/analysis/result.json?id=...`를 fetch해서 렌더링한다.
- `YoutubeCommentAnalysisController`에 분석결과 화면 라우트 `GET /users/youtubeComment/analysis/result/view?id=...`를 추가했다.
- 목록 화면(`youtubeCommentAnalysis.html`)에 `analysisRequestedAt` 컬럼 및 `result` 액션 링크를 추가했다.

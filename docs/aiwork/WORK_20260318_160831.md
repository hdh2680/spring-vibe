# 작업 노트 (AI)

## goal
- PRD 4. 유튜브 댓글분석의 “저장 버튼” 요구사항을 구현한다.
- 저장 버튼 클릭 시 `app.storage.attachments-dir/youtubeComment/yyyyMMdd_HHmmss/comments.jsonl` 경로에 전체 댓글을 끝까지 수집하여 JSON Lines로 저장하고, 저장된 파일을 다운로드로 내려준다.

## key changes
- `YoutubeCommentService`에 전체 댓글을 `pageToken` 기반으로 끝까지 반복 호출하며 JSONL로 스트리밍 저장하는 `exportAllCommentsByUrlAsJsonl()`를 추가했다.
- `YoutubeCommentController`에 `/users/youtubeComment/export` 엔드포인트를 추가해서 저장된 `comments.jsonl`을 `application/x-ndjson` 다운로드로 반환한다.
- 화면에서 `저장(JSONL)` 버튼을 추가했다(검색 폼 값(url/limit/pageToken)을 같이 전송).
- 단위 테스트에서 JSONL 파일이 페이지를 넘겨가며 누적 저장되는지 검증했다.

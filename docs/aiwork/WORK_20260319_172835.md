# 목표 (goal)
분석 결과 화면에 전처리 탭을 추가하고, `analysis_requested_at` 값에 따라 분석 탭(키워드 분석)을 활성/비활성화한다. 또한 기존 전처리 상세 화면(`youtubeCommentHistoryView.html`)을 제거하고 단일 화면(`youtubeCommentAnalysisView.html`)으로 통합한다.

# 핵심 변경사항 (key changes)
- `youtubeCommentAnalysisView.html` 탭 구성 변경: 전처리(첫 탭) / 키워드 분석 / 원본 JSON.
- `analysis_requested_at`이 null이면 키워드 분석 탭은 비활성화되고, 전처리 탭에서 분석수행을 유도한다.
- `YoutubeCommentAnalysisController`의 `/analysis/view`가 `youtubeCommentAnalysisView.html`을 렌더링하도록 유지하고, 기존 `/analysis/result/view`는 `/analysis/view`로 리다이렉트하도록 변경했다.
- `youtubeCommentHistoryView.html` 템플릿을 삭제하고, 목록 화면의 `view` 링크는 항상 `/analysis/view`로 이동하도록 정리했다.

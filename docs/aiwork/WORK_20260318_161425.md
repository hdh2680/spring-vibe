# 작업 노트 (AI)

## goal
- 유튜브 댓글 저장 파일을 `comments.jsonl` 고정 파일명이 아니라, 저장 시각 기반 파일명 `yyyyMMdd_HHmmss.jsonl`로 생성되게 변경한다.
- 저장 경로는 `app.storage.attachments-dir/youtubeComment/{yyyyMMdd_HHmmss}.jsonl` 형식으로 맞춘다.

## key changes
- JSONL 저장 경로를 “타임스탬프 폴더 + comments.jsonl”에서 “youtubeComment 폴더 아래 타임스탬프 파일(yyyyMMdd_HHmmss.jsonl)”로 변경했다.
- 다운로드 응답의 파일명도 저장된 파일명과 동일하게 변경했다.
- 단위 테스트의 경로/파일명 패턴 검증을 업데이트했다.

- `src/main/java/springVibe/dev/users/youtubeComment/service/YoutubeCommentService.java`

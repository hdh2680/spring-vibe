# 목표 (goal)
유튜브 댓글 전처리에서 반복 문자 축약(예: `ㅋㅋㅋㅋ` -> `ㅋㅋ`)이 특정 입력에서 동작하지 않는 문제를 해결한다.

# 핵심 변경사항 (key changes)
- `Normalizer.Form.NFKC` 적용 시 `ㅋ/ㅎ/ㅠ/ㅜ` 호환 자모가 `ᄏ/ᄒ/ᅲ/ᅮ`로 변환되어 기존 반복 축약 정규식에 매칭되지 않는 케이스를 확인했다.
- 전처리 초기에 `ᄏ->ㅋ`, `ᄒ->ㅎ`, `ᅲ->ㅠ`, `ᅮ->ㅜ`로 되돌리는 매핑을 추가해 반복 축약 로직이 정상 적용되도록 수정했다.

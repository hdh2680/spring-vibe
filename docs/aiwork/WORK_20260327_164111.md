# WORK_20260327_164111

## Goal
- 왼쪽 사이드바에 챗봇 진입 버튼을 추가하고, 로컬 Ollama 챗 UI를 페이지 이동 없이 모달로 띄운다.

## Key Changes
- `layout/app.html`에 전역 챗 모달을 추가하고, 사이드바에서 열 수 있는 버튼을 추가했다.
- 사이드바 진입부가 메뉴 링크처럼 보이지 않도록 툴 버튼 형태(아이콘/배지/구분선)로 스타일을 정리했다.
- 모달 UI를 세로로 긴 폰 메신저 느낌(버블/헤더/입력바)으로 리디자인했다.
- `chat.js`를 모달 open/close(Backdrop/ESC 포함) 지원하도록 리팩터링하고, 모달이 없는 페이지(예: login)에서는 no-op 되게 처리했다.
- `/users/chat` 페이지는 DOM id 충돌 방지를 위해 안내/모달 오픈 형태로 변경했다.

## Touched Files
- `src/main/resources/templates/layout/app.html`
- `src/main/resources/static/css/layout_app.css`
- `src/main/resources/static/css/chat.css`
- `src/main/resources/static/js/chat.js`
- `src/main/resources/templates/html/users/chat/chat.html`

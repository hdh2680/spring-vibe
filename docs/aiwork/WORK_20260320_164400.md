# 목표 (goal)
- 감정분석 화면 상단에 전체 건수 대비 긍정/중립/부정/미분류 건수를 설명으로 표시한다.
- “미분류는 시각화에 표현되지 않습니다.” 안내를 추가하고, 실제 차트(파이/시간별 변화)에서는 미분류를 제거한다.
- 감정분포별 댓글 탭 순서를 `긍정/중립/부정/미분류`로 정렬한다.

- 시간별 변화 그래프에서 `unknownPct` 라인을 제거하고, 레전드에서도 미분류를 제거한다.
- 예시 댓글 탭 순서를 `긍정/중립/부정/미분류`로 변경한다.

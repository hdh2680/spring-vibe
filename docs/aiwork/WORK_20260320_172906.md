# 목표 (goal)
- 유튜브 댓글 분석 결과 화면에 **네트워크 분석(Word Co-occurrence Network)** 탭을 추가하고, 분석 수행 시 결과를 DB에 적재한 뒤 화면에서 시각화/표시한다.

# 핵심 변경사항 (key changes)
- DB 테이블 `youtube_comment_analysis_word_networks`를 추가하고, 분석 결과(JSON)를 적재한다.
- MyBatis 매퍼에 네트워크 결과 insert/select/delete를 추가했다.
- 분석 수행(`analyzeAndPersist`) 시 네트워크 결과를 함께 생성/삭제/재적재하도록 연결했다.
- 결과 화면(`youtubeCommentAnalysisView.html`)에 `네트워크 분석` 탭을 추가하고,
  - 간단한 force-layout 기반 SVG 그래프
  - Top Nodes/Top Edges 테이블을 렌더링한다.

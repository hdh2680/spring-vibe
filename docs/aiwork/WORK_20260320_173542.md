# 목표 (goal)
- 네트워크 분석 그래프가 “네트워크처럼 안 보이고 사각형처럼 보인다”는 이슈를 완화한다.
- 예시 데이터에서 보였던 조사/기능어(예: `하고`, `이게`, `있는`, `때문에`)와 형태 변형(예: `미국/미국이/미국은/미국의`)을 줄여 네트워크 품질을 개선한다.

# 핵심 변경사항 (key changes)
- 키워드 토큰 정규화 개선:
  - 한국어 조사/접미(간이) 제거(`미국이` -> `미국`, `트럼프가` -> `트럼프`, `한국에` -> `한국`)
  - 네트워크/키워드 품질에 방해되는 기능어를 stopwords에 추가
- 네트워크 그래프 렌더링 개선:
  - force-layout 파라미터 튜닝(과도한 반발력 감소, 중심 끌어당김/스프링 강화)
  - collision(원형 충돌) 처리를 추가해 노드 겹침을 줄임
  - docFreq에 따라 노드 fill 강도를 약간 다르게 표시

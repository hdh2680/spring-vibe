# 목표 (goal)
- 프로젝트의 화면에서 표시되는 날짜/시간 포맷을 `yyyy-MM-dd HH:mm:ss`로 통일한다.
- 시간 포맷 규칙을 `docs/CODING_RULE.md`에 명문화한다.

# 핵심 변경사항 (key changes)
- `youtubeCommentAnalysisView.html`의 이력 정보(`analysisRequestedAt`)를 `#temporals.format(..., 'yyyy-MM-dd HH:mm:ss')`로 포맷한다.
- `youtubeCommentSearch.html`에서 댓글 `Published`를 `publishedAtView`(포맷된 문자열)로 출력하도록 변경한다.
- 감정분석 결과(JSON)에서 예시 댓글 `publishedAt` 및 시간별 버킷 `hour`를 `yyyy-MM-dd HH:mm:ss` 문자열로 적재한다.
- `docs/CODING_RULE.md`에 날짜/시간 표기 규칙을 추가한다.

# 목표 (goal)
유튜브 댓글 전처리 결과에 `ᄃ` 같은 “한글 자모(ᄀ..ᇂ)” 문자가 노출되는 문제를 방지하고, `ㄷㄷㄷ` 등 과도 반복이 의미 보존 형태로 축약되도록 한다.

# 핵심 변경사항 (key changes)
- `Normalizer.Form.NFKC` 이후 `ㄷ` 등의 호환 자모가 `ᄃ`(Hangul Jamo)로 변환되는 케이스를 일반화해서 처리했다.
- `ᄀ..ᄒ`(초성), `ᅡ..ᅵ`(중성), `ᆨ..ᇂ`(종성) 영역을 호환 자모(ㄱ..ㅎ, ㅏ..ㅣ 등)로 되돌리는 매핑을 추가했다.
- 호환 자모 반복(ㄱ..ㅎ/ㅏ..ㅣ)이 3회 이상 반복될 때 2회로 축약하는 규칙을 추가했다: 예) `ㄷㄷㄷ` -> `ㄷㄷ`.

# 목표 (goal)
유튜브 댓글 분석 목록 화면에서 결과 이동 UX를 단순화한다.

# 핵심 변경사항 (key changes)
- `youtubeCommentAnalysis.html` 목록 화면에서 별도의 `[result]` 링크를 제거했다.
- `view` 클릭 시 `analysis_requested_at` 값에 따라 이동 경로를 분기했다.
  - NULL: 전처리 상세 화면 `youtubeCommentHistoryView.html`로 이동
  - NOT NULL: 분석 결과 화면 `youtubeCommentAnalysisView.html`로 이동

# 수정/추가한 파일 (touched files)
- `src/main/resources/templates/html/users/youtubeComment/youtubeCommentAnalysis.html`

# 실행/테스트한 명령 (run/test commands)
- `.\mvnw.cmd -q -DskipTests compile`

# 후속 할 일 (follow-ups)
- 목록 화면에서 `analysisRequestedAt` 컬럼 표시 포맷(yyyy-MM-dd HH:mm 등) 적용 여부 검토.

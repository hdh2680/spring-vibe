# PROJECT_CONTEXT.md

## Project Overview

이 프로젝트는 AI와 협업하여 개발하는 백엔드 프로젝트이다.

개발자는 설계자 역할을 하며,
AI는 문서를 기반으로 기능을 구현한다.

모든 개발 작업은 문서 기반으로 진행된다.

---

## Important Documents

AI는 아래 문서를 기준으로 작업해야 한다.

PRD.md
→ 프로젝트 요구사항

DATABASE_SCHEMA.md
→ 데이터베이스 구조

ARCHITECTURE.md
→ 프로젝트 구조

CODING_RULE.md
→ 코딩 규칙

FUNCTIONS.md
→ 구현된 기능 목록

TODOLIST.md
→ 작업 목록

---

## Development Workflow

개발 절차

1. PRD 확인
2. DATABASE_SCHEMA 확인
3. ARCHITECTURE 확인
4. CODING_RULE 확인
5. 기능 구현
6. FUNCTIONS.md 업데이트
7. 작업 로그 docs/aiwork 기록

---

## Important Rules

- PRD 기반으로 기능을 구현한다.
- DATABASE_SCHEMA를 기준으로 Entity를 작성한다.
- CODING_RULE를 반드시 따른다.
- 작업 완료 후 FUNCTIONS.md를 업데이트한다.
